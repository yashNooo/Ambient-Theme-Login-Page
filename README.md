# Ambient-Theme-Login-Page
## A Basic Login Page that have a attractive ambient light turn on by a switch and make the password visible.

## Switch on => Password Visible and ambient light on
## Switch of => Password Hide and ambient light of

### We can type our mail and enter the password. Responsive ui with email format checker.
